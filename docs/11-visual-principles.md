# Desklabs Visual Principles

## V001

Rhythm over Borders.

Separate sections using spacing first.

Borders only when necessary.

---

## V002

Typography creates hierarchy.

Color only reinforces hierarchy.

---

## V003

Whitespace is structure.

Never use empty cards to create breathing room.

---

## V004

Every page should feel like one continuous workspace.

Not stacked widgets.

---

## V005

One focal point.

Users should always know where to look first.

---

## V006

Quiet by default.

Highlight only what requires attention.

---

## V007

Density with clarity.

Show more work without feeling crowded.

---

## V008

Every pixel should justify its existence.
