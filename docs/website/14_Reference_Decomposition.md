# Supplied Reference Decomposition

## Purpose

The supplied static page and motion video are a benchmark, not a template to copy. This document translates the useful design principles into original Desklabs decisions.

## What works in the reference

### 1. Product-led hero

The hero communicates through a product scene rather than relying only on headline claims. Floating callouts demonstrate outcomes while the main UI establishes credibility.

**Desklabs adaptation:** use a desktop customer-operations workspace, not the reference mobile phone composition.

### 2. Narrative order

The page follows a coherent journey:

Hero → industries → common problems → unified platform → how it works → proof → CTA.

**Desklabs adaptation:** insert a platform-core section that explains multi-industry architecture and a dedicated Aurora AI story.

### 3. Alternating problem rows

The repetition is easy to scan while each row remains focused.

**Desklabs adaptation:** four universal operational problems with original Desklabs UI states.

### 4. Restrained color

White space dominates. Blue/purple gradients frame important moments rather than covering the entire page.

**Desklabs adaptation:** Brand Blue, Aurora Violet, and Cyan glows with more neutral product surfaces.

### 5. Ecosystem visual

Channel icons and connected messages make “one platform” visually understandable.

**Desklabs adaptation:** connect communication channels with CRM, workflow, finance, and AI, not only messaging.

### 6. Calm motion

Motion uses fade, translate, blur, and stagger. It does not rely on theatrical 3D effects.

**Desklabs adaptation:** motion explains customer/workflow state changes and always supports reduced-motion.

## What must not be copied

- Exact hero composition
- Exact text or headline structure
- Exact purple palette
- Illustration assets
- Customer logos or proof
- Phone mockup arrangement
- Section dimensions and card geometry frame-for-frame
- Animation sequence timing frame-for-frame
- Icon treatment unique to the reference brand

## Original Desklabs differentiators

- Platform core plus vertical operating templates
- Customer operations, not only omnichannel messaging
- Aurora human-in-the-loop AI
- Communication + CRM + operations + finance in one model
- Indonesian-first conversion copy
- Industry pages that map real business lifecycles
- Calm Workspace visual continuity between marketing site and application
